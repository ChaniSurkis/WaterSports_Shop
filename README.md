# WaterSports Shop  
### E-Commerce Platform for Water Sports Equipment

WaterSports Shop is a full-stack e-commerce web application for selling water sports products.  
The project includes both client-side and admin functionality, simulating a real commercial online store.

**This project is currently under active development.**

---

## Overview
The purpose of this project is to demonstrate the development of a modern e-commerce system using contemporary web technologies.  
It focuses on state management, routing, and separation between client and admin flows.

---

## Main Features

### Client Side
- Product listing and product details
- Shopping cart management
- Add / remove products from cart
- Client navigation using React Router
- Global state management for cart and user data

### Admin Side
- Admin dashboard
- Product management (add / update / remove products)
- Basic control over store data

---

## Technologies Used

### Frontend
- React

### Backend
